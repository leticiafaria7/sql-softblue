# sql-softblue
Repositório para colocar os arquivos referentes ao curso de SQL da softblue

O curso gratuito de SQL da softblue pode ser acessado [neste link](https://www.softblue.com.br/site/curso/id/3/CURSO+SQL+COMPLETO+BASICO+AO+AVANCADO+ON+LINE+BD03)

Conteúdo do curso:

- Módulo 1: Introdução a bancos de dados
- Módulo 2: Normalização de dados
- Módulo 3: Criando um banco de dados
- Módulo 4: Manipulando dados
- Módulo 5: Relacionamento e visões
- Módulo 6: Funções especiais e subqueries
- Módulo 7: Controle de acesso 
- Módulo 8: Transações (ACID)
- Módulo 9: Stored procedures e triggers
